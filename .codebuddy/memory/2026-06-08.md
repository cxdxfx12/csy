## 登录成功但页面不跳转问题
- 用户报告"显示登录成功，但是页面没有跳转"
- 根因：服务器上缺少前端构建文件 `index-DhNUDoAr.js` 和 `RoleIcon.vue_vue_type_script_setup_true_lang-DdRNPsNZ.js`
- `index-DhNUDoAr.js` 是 Vite 构建的共享 chunk（Vue 运行时等），被 `DashboardView` 动态引用
- 缺少此文件导致 Dashboard 组件加载失败（404），页面停留在 login 不跳转
- 修复：重新 `npm run build` 并将所有 240 个 assets 文件完整同步到服务器
- 验证通过：登录后成功跳转到 /admin/dashboard，页面标题"控制台"正常显示
